# Шаблоны

## Отправка с шаблоном

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

# По ID шаблона
response = client.send_email_with_template({
    "TemplateId": 42,
    "From": "noreply@example.com",
    "To": "user@example.com",
    "TemplateModel": {
        "product_name": "Мой Сервис",
        "user_name": "Иван",
    },
})

# По алиасу шаблона
response = client.send_email_with_template({
    "TemplateAlias": "welcome",
    "From": "noreply@example.com",
    "To": "user@example.com",
    "TemplateModel": {"user_name": "Иван"},
})
```

## Список шаблонов

```python
templates = client.get_templates(count=100, offset=0)

for t in templates["Templates"]:
    print(f"[{t['TemplateId']}] {t['Name']} — {t.get('Alias', 'без алиаса')}")
```

### Параметры фильтрации

| Параметр | Тип | Описание |
|----------|-----|----------|
| `count` | int | Макс. количество (до 300) |
| `offset` | int | Смещение для пагинации |
| `TemplateType` | string | `"Standard"` или `"Layout"` |

## Получить шаблон

```python
# По ID
template = client.get_template(42)

# По алиасу
template = client.get_template("welcome")

print(template["Name"])
print(template["Subject"])
print(template["HtmlBody"])
```

## Создать шаблон

```python
template = client.create_template({
    "Name": "Сброс пароля",
    "Alias": "password-reset",
    "Subject": "Сброс пароля для {{product_name}}",
    "HtmlBody": "<h1>Сбросить пароль</h1><p>Нажмите <a href='{{action_url}}'>сюда</a>.</p>",
    "TextBody": "Перейдите по ссылке: {{action_url}}",
})

print(f"Создан шаблон ID: {template['TemplateId']}")
```

## Редактировать шаблон

```python
client.edit_template(42, {
    "Name": "Сброс пароля v2",
    "Subject": "Новый пароль для {{product_name}}",
})
```

## Удалить шаблон

```python
client.delete_template(42)
# или по алиасу:
client.delete_template("password-reset")
```

## Валидировать шаблон

Проверить шаблон без создания:

```python
result = client.validate_template({
    "Subject": "Привет, {{name}}!",
    "HtmlBody": "<h1>{{greeting}}</h1>",
    "TextBody": "{{greeting}}",
    "TestRenderModel": {
        "name": "Тест",
        "greeting": "Привет!",
    },
})

print(f"Валидно: {result['AllContentIsValid']}")
print(f"HTML результат: {result['HtmlBody']['ContentIsValid']}")
print(f"Рендер HTML: {result['HtmlBody']['RenderedContent']}")
```

## Push шаблонов между серверами

Для переноса шаблонов используйте `AccountClient`:

```python
from haskimail import AccountClient

account = AccountClient("ваш-аккаунт-токен")

result = account.push_templates({
    "SourceServerID": 1,
    "DestinationServerID": 2,
    "PerformChanges": True,
})
```
