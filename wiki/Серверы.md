# Серверы

Управление серверами доступно через `AccountClient`.

```python
from haskimail import AccountClient

account = AccountClient("ваш-аккаунт-токен")
```

## Список серверов

```python
servers = account.get_servers(count=50, offset=0)

for s in servers["Servers"]:
    print(f"[{s['ID']}] {s['Name']} ({s['Color']})")
```

## Получить сервер

```python
server = account.get_server(12345)
print(f"Имя: {server['Name']}")
print(f"SMTP активирован: {server['SmtpApiActivated']}")
```

## Создать сервер

```python
server = account.create_server({
    "Name": "Production",
    "Color": "Green",
})

print(f"Создан сервер ID: {server['ID']}")
print(f"API-токены: {server['ApiTokens']}")
```

### Цвета серверов

`Purple`, `Blue`, `Turqoise`, `Green`, `Red`, `Orange`, `Yellow`, `Grey`

## Редактировать сервер

```python
account.edit_server(server["ID"], {
    "Name": "Production v2",
    "Color": "Blue",
})
```

## Удалить сервер

```python
account.delete_server(server["ID"])
```

> Удаление через API требует специального разрешения. Обратитесь в поддержку, если получаете ошибку 604.
