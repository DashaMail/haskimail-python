# Открытия и клики

## Открытия

### Все открытия

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

opens = client.get_message_opens(
    count=50,
    offset=0,
    recipient="user@example.com",
)

for event in opens["Opens"]:
    print(f"{event['Recipient']} открыл в {event['ReceivedAt']}")
    print(f"  Клиент: {event['Client']['Name']} ({event['OS']['Name']})")
```

### Открытия конкретного сообщения

```python
opens = client.get_message_opens_for_single_message(
    "b7bc2f4a-e38e-4336-af7d-e6c392c2f817",
    count=50,
)

for event in opens["Opens"]:
    print(f"Открытие в {event['ReceivedAt']} с {event['Client']['Name']}")
```

### Параметры фильтрации

| Параметр | Тип | Описание |
|----------|-----|----------|
| `count` | int | Макс. количество (до 500) |
| `offset` | int | Смещение |
| `recipient` | string | Фильтр по получателю |
| `tag` | string | Фильтр по тегу |
| `client_name` | string | Фильтр по клиенту (Gmail, Outlook, ...) |
| `client_company` | string | Компания email-клиента |
| `client_family` | string | Семейство клиента |
| `os_name` | string | ОС |
| `os_family` | string | Семейство ОС |
| `os_company` | string | Компания ОС |
| `platform` | string | Платформа (Desktop, Mobile, WebMail) |
| `country` | string | Страна |
| `region` | string | Регион |
| `city` | string | Город |
| `messagestream` | string | ID канала |

## Клики

### Все клики

```python
clicks = client.get_message_clicks(
    count=50,
    recipient="user@example.com",
    tag="newsletter",
)

for event in clicks["Clicks"]:
    print(f"{event['Recipient']} кликнул {event['OriginalLink']}")
    print(f"  Браузер: {event['Client']['Name']}, Платформа: {event['Platform']}")
```
