# Установка и настройка

## Установка

```bash
pip install haskimail
```

## Требования

- Python 3.10+
- requests >= 2.28.0

## Создание клиента

### ServerClient

Для отправки писем и операций на уровне сервера:

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")
```

Серверный токен можно найти в настройках сервера → вкладка «API-токены» в [панели управления Haskimail](https://lk.haskimail.ru).

### AccountClient

Для управления серверами, доменами и подписями:

```python
from haskimail import AccountClient

account = AccountClient("ваш-аккаунт-токен")
```

Аккаунт-токен доступен в [настройках аккаунта → API-ключи](https://lk.haskimail.ru/?page=account&action=api_tokens).

## Параметры

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `base_url` | `https://api.haskimail.ru` | Базовый URL API |
| `timeout` | `30` | Таймаут HTTP-запросов (секунды) |

```python
client = ServerClient(
    "ваш-токен",
    base_url="https://api.haskimail.ru",
    timeout=60,
)
```

## Context Manager

Клиент можно использовать как контекстный менеджер:

```python
with ServerClient("ваш-токен") as client:
    client.send_email({...})
# HTTP-сессия закрывается автоматически
```

## Тестовый режим

Для тестирования без реальной отправки:

```python
client = ServerClient("HASKIMAIL_API_TEST")
response = client.send_email({...})  # Валидирует, но не отправляет
```
