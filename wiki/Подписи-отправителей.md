# Подписи отправителей (Sender Signatures)

Управление подписями доступно через `AccountClient`.

```python
from haskimail import AccountClient

account = AccountClient("ваш-аккаунт-токен")
```

## Список подписей

```python
signatures = account.get_sender_signatures(count=100, offset=0)

for s in signatures["SenderSignatures"]:
    confirmed = "✓" if s["Confirmed"] else "✗"
    print(f"[{s['ID']}] {s['Name']} <{s['EmailAddress']}> {confirmed}")
```

## Получить подпись

```python
sig = account.get_sender_signature(42)
print(f"Имя: {sig['Name']}")
print(f"Email: {sig['EmailAddress']}")
print(f"Подтверждена: {sig['Confirmed']}")
```

## Создать подпись

```python
sig = account.create_sender_signature({
    "Name": "Отдел продаж",
    "FromEmail": "sales@example.com",
    "ReplyToEmail": "support@example.com",
})

print(f"Создана подпись ID: {sig['ID']}")
print(f"Подтверждена: {sig['Confirmed']}")
```

## Редактировать подпись

```python
account.edit_sender_signature(sig["ID"], {
    "Name": "Команда продаж",
    "ReplyToEmail": "team@example.com",
})
```

## Повторно отправить подтверждение

```python
account.resend_sender_signature_confirmation(sig["ID"])
```

## Проверить SPF

```python
result = account.verify_sender_signature_spf(sig["ID"])
print(f"SPF verified: {result['SPFVerified']}")
```

## Запросить новый DKIM

```python
account.request_new_dkim_for_sender_signature(sig["ID"])
```

## Удалить подпись

```python
account.delete_sender_signature(sig["ID"])
```
