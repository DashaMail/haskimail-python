# Исходящие сообщения

## Список исходящих

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

messages = client.get_outbound_messages(
    count=50,
    offset=0,
    recipient="user@example.com",
    tag="welcome",
)

for msg in messages["Messages"]:
    print(f"[{msg['MessageID']}] {msg['Subject']} → {msg['To'][0]['Email']}")
```

### Параметры фильтрации

| Параметр | Тип | Описание |
|----------|-----|----------|
| `count` | int | Макс. количество (до 500) |
| `offset` | int | Смещение |
| `recipient` | string | Фильтр по получателю |
| `fromemail` | string | Фильтр по отправителю |
| `tag` | string | Фильтр по тегу |
| `subject` | string | Фильтр по теме |
| `status` | string | Статус |
| `fromdate` | string | Дата начала |
| `todate` | string | Дата конца |
| `messagestream` | string | ID канала |

## Детали сообщения

```python
details = client.get_outbound_message_details("b7bc2f4a-e38e-4336-af7d-e6c392c2f817")

print(f"Тема: {details['Subject']}")
print(f"Статус: {details['Status']}")
print(f"Получатели: {[r['Email'] for r in details['To']]}")
```
