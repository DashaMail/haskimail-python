# Обработка ошибок

## Иерархия исключений

```
HaskimailError                  — базовое исключение SDK
└── HttpError                   — ошибка HTTP-уровня
    ├── InvalidAPIKeyError      — 401 (неверный или отсутствующий токен)
    ├── ApiInputError           — 422 (ошибка в данных запроса)
    │   ├── InactiveRecipientsError   — 422/406 (неактивный получатель)
    │   └── InvalidEmailRequestError  — 422/300 (неверный формат email)
    ├── RateLimitExceededError  — 429 (превышен лимит запросов)
    ├── InternalServerError     — 500 (ошибка сервера)
    ├── ServiceUnavailableError — 503 (сервис недоступен)
    └── UnknownError            — другие HTTP-ошибки
```

## Свойства HttpError

| Свойство | Тип | Описание |
|----------|-----|----------|
| `message` | str | Текст ошибки от API |
| `status_code` | int | HTTP-код ответа |
| `error_code` | int | ErrorCode от API (специфичный код ошибки) |
| `body` | dict \| None | Полное тело ответа |

## InactiveRecipientsError

Дополнительное свойство:

| Свойство | Тип | Описание |
|----------|-----|----------|
| `recipients` | list[str] | Список неактивных email-адресов |

## Примеры

### Обработка конкретных ошибок

```python
from haskimail import (
    ServerClient,
    InactiveRecipientsError,
    InvalidAPIKeyError,
    RateLimitExceededError,
    ApiInputError,
    HttpError,
)

client = ServerClient("ваш-токен")

try:
    client.send_email({
        "From": "sender@example.com",
        "To": "recipient@example.com",
        "Subject": "Тест",
        "TextBody": "Привет!",
    })
except InactiveRecipientsError as e:
    print(f"Неактивные адреса: {', '.join(e.recipients)}")

except InvalidAPIKeyError:
    print("Проверьте API-токен!")

except RateLimitExceededError:
    import time
    time.sleep(10)  # Подождите и повторите

except ApiInputError as e:
    print(f"Ошибка валидации: {e.message} (ErrorCode: {e.error_code})")

except HttpError as e:
    print(f"HTTP {e.status_code}: {e.message}")
```

### Получение полного тела ошибки

```python
try:
    client.send_email({})
except HttpError as e:
    print(e.body)  # {"ErrorCode": 300, "Message": "..."}
```

## Коды ошибок API

Полный список кодов ошибок доступен в [документации API](https://haskimail.ru/developer/api/overview#error-codes).

Наиболее частые:

| Код | Описание |
|-----|----------|
| 10 | Неверный или отсутствующий API-токен |
| 300 | Неверный запрос на отправку письма |
| 400 | Подпись отправителя не найдена |
| 401 | Подпись отправителя не подтверждена |
| 405 | Недостаточно кредитов |
| 406 | Неактивный получатель |
| 409 | Требуется JSON (заголовки Accept/Content-Type) |
| 410 | Слишком много сообщений в пакете (> 500) |
