# Статистика

Все методы статистики принимают одинаковые фильтры:

| Параметр | Тип | Описание |
|----------|-----|----------|
| `tag` | string | Фильтр по тегу |
| `fromdate` | string | Дата начала (ISO 8601) |
| `todate` | string | Дата конца |
| `messagestream` | string | ID канала |

## Общий обзор

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

overview = client.get_outbound_overview(
    fromdate="2026-01-01",
    todate="2026-06-01",
)

print(f"Отправлено: {overview['Sent']}")
print(f"Возвраты: {overview['Bounced']}")
print(f"Жалобы на спам: {overview['SpamComplaints']}")
print(f"Открытия: {overview['Opens']}")
print(f"Уникальные открытия: {overview['UniqueOpens']}")
print(f"Клики: {overview['TotalClicks']}")
```

## Отправки по дням

```python
sent = client.get_sent_counts(tag="welcome")
for day in sent["Days"]:
    print(f"{day['Date']}: {day['Sent']}")
```

## Возвраты по дням

```python
bounces = client.get_bounce_counts()
for day in bounces["Days"]:
    print(f"{day['Date']}: Hard={day['HardBounce']}, Soft={day['SoftBounce']}")
```

## Открытия по email-клиентам

```python
clients = client.get_email_open_client_usage()
for c in clients["Days"]:
    for client_info in c.get("Clients", []):
        print(f"  {client_info['Name']}: {client_info['Count']}")
```
