# Вебхуки

## Список вебхуков

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

webhooks = client.get_webhooks(MessageStream="outbound")

for wh in webhooks["Webhooks"]:
    print(f"[{wh['ID']}] {wh['Url']}")
    triggers = wh.get("Triggers", {})
    for trigger_name, trigger_config in triggers.items():
        if trigger_config.get("Enabled"):
            print(f"  ✓ {trigger_name}")
```

## Получить вебхук

```python
webhook = client.get_webhook(42)
print(f"URL: {webhook['Url']}")
```

## Создать вебхук

```python
webhook = client.create_webhook({
    "Url": "https://example.com/webhooks/haskimail",
    "MessageStream": "outbound",
    "Triggers": {
        "Open": {"Enabled": True, "PostFirstOpenOnly": False},
        "Click": {"Enabled": True},
        "Delivery": {"Enabled": True},
        "Bounce": {"Enabled": True, "IncludeContent": True},
        "SpamComplaint": {"Enabled": True, "IncludeContent": True},
        "SubscriptionChange": {"Enabled": False},
    },
    "HttpHeaders": [
        {"Name": "X-Webhook-Secret", "Value": "my-secret-key"},
    ],
})

print(f"Создан вебхук ID: {webhook['ID']}")
```

## Редактировать вебхук

```python
client.edit_webhook(webhook["ID"], {
    "Url": "https://example.com/webhooks/haskimail-v2",
    "Triggers": {
        "Open": {"Enabled": True, "PostFirstOpenOnly": True},
    },
})
```

## Удалить вебхук

```python
client.delete_webhook(webhook["ID"])
```

## Типы триггеров

| Триггер | Описание |
|---------|----------|
| `Open` | Открытие письма |
| `Click` | Клик по ссылке |
| `Delivery` | Успешная доставка |
| `Bounce` | Возврат |
| `SpamComplaint` | Жалоба на спам |
| `SubscriptionChange` | Изменение подписки |
