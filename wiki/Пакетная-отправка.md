# Пакетная отправка

## send_email_batch

Отправка до 500 уникальных писем одним API-вызовом:

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

responses = client.send_email_batch([
    {
        "From": "noreply@example.com",
        "To": "user1@example.com",
        "Subject": "Привет, User 1!",
        "TextBody": "Персональное сообщение для User 1",
        "Tag": "batch-notification",
        "MessageStream": "outbound",
    },
    {
        "From": "noreply@example.com",
        "To": "user2@example.com",
        "Subject": "Привет, User 2!",
        "TextBody": "Персональное сообщение для User 2",
        "Tag": "batch-notification",
        "MessageStream": "outbound",
    },
])

for resp in responses:
    if resp.get("ErrorCode", 0) == 0:
        print(f"✓ {resp['To']} → {resp['MessageID']}")
    else:
        print(f"✗ {resp['To']} → {resp['Message']}")
```

> **Важно:** Пакетный запрос возвращает HTTP 200 даже если отдельные письма не прошли. Проверяйте `ErrorCode` каждого элемента.

## send_email_batch_with_templates

Пакетная отправка с использованием шаблонов:

```python
responses = client.send_email_batch_with_templates([
    {
        "TemplateAlias": "welcome",
        "From": "noreply@example.com",
        "To": "user1@example.com",
        "TemplateModel": {"name": "User 1", "plan": "Pro"},
        "MessageStream": "outbound",
    },
    {
        "TemplateAlias": "welcome",
        "From": "noreply@example.com",
        "To": "user2@example.com",
        "TemplateModel": {"name": "User 2", "plan": "Free"},
        "MessageStream": "outbound",
    },
])
```

## Ограничения

| Параметр | Лимит |
|----------|-------|
| Писем в пакете | 500 |
| Размер запроса (Email API) | 10 МБ |
| Размер запроса (Batch API) | 50 МБ |
