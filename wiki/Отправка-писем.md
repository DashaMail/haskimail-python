# Отправка писем

## Отправка одного письма

```python
from haskimail import ServerClient

client = ServerClient("ваш-серверный-токен")

response = client.send_email({
    "From": "John Doe <john@example.com>",
    "To": "recipient@example.com",
    "Subject": "Подтверждение заказа",
    "HtmlBody": "<h1>Ваш заказ подтверждён!</h1>",
    "TextBody": "Ваш заказ подтверждён!",
})

print(response["MessageID"])      # "b7bc2f4a-e38e-4336-af7d-e6c392c2f817"
print(response["SubmittedAt"])    # "2026-06-01T12:00:00.0000000+00:00"
```

## Параметры email

| Поле | Тип | Обязательно | Описание |
|------|-----|-------------|----------|
| `From` | string | Да | Отправитель: `"email"` или `"Имя <email>"` |
| `To` | string | Да | Получатель(и), через запятую (макс. 50) |
| `Cc` | string | Нет | Копия |
| `Bcc` | string | Нет | Скрытая копия |
| `Subject` | string | Да | Тема |
| `HtmlBody` | string | Да* | HTML-версия (*нужен HtmlBody или TextBody) |
| `TextBody` | string | Да* | Текстовая версия |
| `Tag` | string | Нет | Тег для группировки (фильтрация в статистике) |
| `ReplyTo` | string | Нет | Адрес для ответа |
| `Headers` | list | Нет | Пользовательские заголовки |
| `TrackOpens` | bool | Нет | Отслеживать открытия |
| `TrackLinks` | string | Нет | `"None"`, `"HtmlAndText"`, `"HtmlOnly"`, `"TextOnly"` |
| `Metadata` | dict | Нет | Метаданные (ключ-значение) |
| `MessageStream` | string | Нет | ID канала (по умолчанию `"outbound"`) |
| `Attachments` | list | Нет | Вложения (см. ниже) |

## Вложения

```python
import base64
from pathlib import Path

content = base64.b64encode(Path("invoice.pdf").read_bytes()).decode()

response = client.send_email({
    "From": "billing@example.com",
    "To": "client@example.com",
    "Subject": "Счёт за июнь",
    "TextBody": "Счёт во вложении.",
    "Attachments": [
        {
            "Name": "invoice-june-2026.pdf",
            "Content": content,
            "ContentType": "application/pdf",
        }
    ],
})
```

## Пользовательские заголовки

```python
response = client.send_email({
    "From": "noreply@example.com",
    "To": "user@example.com",
    "Subject": "Сброс пароля",
    "TextBody": "Ваш код: 123456",
    "Headers": [
        {"Name": "X-Custom-ID", "Value": "password-reset-42"},
    ],
})
```

## Метаданные

Метаданные — пары ключ-значение, которые вы можете фильтровать в статистике и вебхуках:

```python
response = client.send_email({
    "From": "noreply@example.com",
    "To": "user@example.com",
    "Subject": "Добро пожаловать",
    "TextBody": "Привет!",
    "Metadata": {
        "user_id": "12345",
        "campaign": "onboarding",
    },
})
```
